# Proxima Dashboard (Frontend)

This is the frontend dashboard for the Proxima Smart Temperature Optimiser.

## Features

- Live system status
- AI & energy insights
- Energy usage visualisation
- Eco-friendly UI

## Tech Stack

- React
- Plain CSS
- Node.js backend (API-based)

This dashboard currently uses simulated data and is designed to integrate
with real-time IoT devices and ML models in future.
